### [ Logical Volume Management ]

  - name: Creating a logical volumes
    desc: Create the logiacal volume /nsr
    command: crfs -v jfs2 -g rootvg  -m /nsr -a size=2G -A yes 
	
  - name: Extend VG filesystem
    desc: Check if server is clustered
    command: lspv
	
    desc: If not clustered, to check available free space
    command:  lsvg rootvg
	
    desc: To show list of logical volumes
    command: ls -l rootvg	
	
    desc: To add additional 2G to /opt
    command: chfs -a size=+2G /opt
